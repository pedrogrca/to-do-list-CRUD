# 📝 Lista de Tarefas (To-Do List)

Uma aplicação web simples, elegante e funcional para o gerenciamento de tarefas do dia a dia. Construída com as tecnologias web fundamentais HTML, CSS e JavaScript (Puro, com o uso de DOM), esta lista permite que os usuários adicionem, concluam e removam atividades com facilidade. 

O grande diferencial é a integração com o **Local Storage**, garantindo que as tarefas permaneçam salvas no navegador mesmo se a página for recarregada.

## ✨ Funcionalidades

* ➕ **Adição de Tarefas:** Interface intuitiva para inserir novas atividades. Prevenção de entradas vazias com alertas na tela.
* ✅ **Status de Conclusão:** Clique em qualquer tarefa para alternar seu status para "concluído" (texto riscado e mudança de ícone).
* ❌ **Remoção Fácil:** Botão de exclusão (`×`) embutido em cada item da lista.
* 💾 **Persistência de Dados:** Uso do `localStorage` do navegador para salvar o estado atual da lista de tarefas automaticamente.
* 🎨 **Design Minimalista:** Estilização limpa utilizando a tipografia moderna *Funnel Sans* do Google Fonts e efeitos de hover suaves.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização avançada, uso de Flexbox para alinhamento e pseudo-elementos (`::before` e `hover`) para criação de checkboxes customizados.
* **JavaScript (Vanilla):** Lógica de manipulação do DOM (Document Object Model) e gerenciamento de dados no navegador a partir de localStorage.
